# *Abhidhānacintāmaṇi-pariśiṣṭa* (`ACPH`)

`ACPH` is the **pariśiṣṭa (supplement)** to Hemacandra's [*Abhidhānacintāmaṇi* (ABCH)](abch) —
a short addendum of further synonym groups in the same versified-kośa form. Read it together
with the main kośa.

## When to use it

Use `ACPH` as the **supplement to [ABCH](abch)** — extra synonym groups not in the main text.
For meanings in English use **[Monier-Williams (MW)](mw)**.

## Reading an entry

Same form as the main kośa: a synonym group (`<syns>`) followed by its verse, within a named
section. The first entry (`csl-orig/v02/acph/acph.txt`):

```
<L>1<pc>1
<info kvvv="<s>devakARqaH</s>"/>
<eid>341<syns><s>Palodaya-puM,merupfzWa-klI,vAsavAvAsa-puM,sErika-puM,…</s>
<s>Palodayo merupfzWaM vAsavAvAsasErikO .</s>
```

| In the source | Meaning |
|---|---|
| `<info kvvv="…devakARqaH">` | the subject section (here "of gods") |
| `<eid>341` | the entry id (continuing the main kośa's numbering) |
| `<syns><s>…</s>` | the **synonyms** (with gender tags `-puM`, `-klI` = masc., neut.) |
| the `<s>…</s>` verse | the **verse** stating them |

See [Data Formats](../developers/data-formats) for the markup reference.
