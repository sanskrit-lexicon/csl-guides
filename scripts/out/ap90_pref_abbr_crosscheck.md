## Residual notes

- Case is folded by default (pref `Bhag.` matches body `BHAG.`).
- Diacritic / OCR orthography is **not** folded in v1 (`Âçv.` ≠ body `ACV.`, `Çâk.` ≠ `SAK.`).

- Compound keys with spaces (`Ait. Br.`, `H. an.`) need the body to use the same spacing.
