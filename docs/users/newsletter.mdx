Monthly updates from the Cologne Digital Sanskrit Dictionaries project — corrections,
new features, documentation, and work in progress.

Past editions are published on [News](/news) and archived at
[github.com/sanskrit-lexicon/csl-newsletter](https://github.com/sanskrit-lexicon/csl-newsletter).

## What to expect

- **Monthly cadence** — one edition per month, sent near the end of the month
- **Coverage** — corrections to the 42+ dictionaries, documentation updates, tool changes,
  and occasional notes on ongoing scholarly work
- **Format** — plain text, always archived here on [/news](/news) and on GitHub
