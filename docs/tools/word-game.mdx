# Sanskrit Word Game

See a word in Devanagari, guess what it means, build your streak. No prior knowledge
needed — the first few words are picked to be easy wins, and every meaning is drawn
from a verified Monier-Williams dictionary entry, not guessed.

<WordGame />

The words come from a **frequency-ranked** vocabulary list — the same
[corpus-frequency data](../developers/data-cards) used across this site — so the
words you meet first here are the ones you'll actually run into most often in real
Sanskrit texts, not an arbitrary sample.

:::tip Enjoying the words?
The [Your Name in Devanagari](name-in-devanagari) tool and the
[What's Your Sanskrit Level?](sanskrit-level-quiz) quiz are good next stops — and the
[Quick Start](../users/quick-start) guide picks up from there.
:::
