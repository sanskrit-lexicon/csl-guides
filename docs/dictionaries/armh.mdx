# Halāyudha, *Abhidhānaratnamālā* (`ARMH`)

`ARMH` is Halāyudha Bhaṭṭa's *Abhidhānaratnamālā* (the *Halāyudha-kośa*) — a classical
**synonym dictionary (kośa)** in verse, edited by **Theodor Aufrecht**. Like other kośas it
gathers the **synonyms of a concept** under a headword, stated in a memorisable verse.

## When to use it

Reach for `ARMH` to find the **classical synonyms** of a word in a second kośa tradition
(alongside [Hemacandra's ABCH](abch)). For English meanings use **[Monier-Williams (MW)](mw)**;
for the large modern kośas see **[SKD](skd)** / **[VCP](vcp)**.

## Reading an entry

Each entry has a **headword** (`<k1>`), a **verse number** (`<vn>`), and the **verse** listing
the synonyms. The first entry — synonyms of *svar* (heaven) (`csl-orig/v02/armh/armh.txt`):

```
<L>1<pc>0001<k1>svar<k2>svar<vn>1.1.1.3
svar;
svaH svargaH surasadma tridaSAvAsastrivizwapaM tridivam .
dyOrgOramartyaBuvanaM nAkaH syAdUrDvalokaSca .. 3 ..
```

| In the source | Meaning |
|---|---|
| `<k1>svar` | the headword (the leading synonym; search key in SLP1) |
| `<vn>1.1.1.3` | the verse number / locus |
| the two `<s>…</s>` lines | the **verse** giving the synonyms — *svaḥ, svarga, surasadman, tridaśāvāsa, triviṣṭapa, tridiva, dyo, …* for "heaven" |

See [Data Formats](../developers/data-formats) for the markup reference.

## What makes it distinctive

- **A second classical synonym kośa**, edited by the noted indologist Theodor Aufrecht.
- **Headword + verse** structure — find a word, get its synonym set.


## Headword key conventions

`<k1>` = `<k2>` = SLP1 (the **leading synonym** of the verse set). Each record corresponds to a **verse** (`<vn>` gives the verse number locus); the full synonym set is encoded in the verse lines.

| Convention | Marker / form | Example | Note |
|---|---|---|---|
| Search key | SLP1 (leading synonym of the set) | `svar` | k1 = k2; the first word of the synonym verse |
| Verse number | `<vn>kāṇḍa.section.chapter.verse` | `<vn>1.1.1.3` | Locates the verse in the text |
| Synonyms | Subsequent words in the verse | `svaH svargaH surasadma …` | All are synonyms for the same concept |
