# Abbreviations compared

Every Cologne dictionary cites the same body of Sanskrit literature and uses the same
grammatical categories — but each prints its **own** abbreviations for them. Monier-Williams
writes the Ṛgveda `RV.`, Benfey `Rigv.`, Vaidya `R.V.`, the Petersburg dictionaries `ṚV.`
This page lines those legends up against each other, and then lists each dictionary's full
legend where it has been transcribed.

<AbbreviationSummary />

The data is generated from the canonical abbreviation files in each dictionary's repository
(see the `source:` link in every section below); it is **not** hand-authored. Where a
dictionary's legend exists only as a scanned page, the section links to that front matter
instead. See [Abbreviations & Citations](abbreviations-and-citations) for how the `<ls>`,
`<ab>` and `<lex>` markup tags carry these abbreviations inside an entry.

How each dictionary abbreviates the major cited **works**. Rows are ordered by how many

## Grammatical &amp; general terms, side by side

The same comparison for **grammatical** abbreviations (gender, case, pada, verb classes).
French (BUR, STC), German (GRA, PWG, PW) and Latin (BOP) dictionaries are matched by the
meaning of the term, so e.g. *accusative* lines up across `acc.`, `Acc.` and `accusatif`.
