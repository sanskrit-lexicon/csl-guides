# Jinadeva, *Abhidhānacintāmaṇi-śiloñcha* (`ACSJ`)

`ACSJ` is the *śiloñcha* ("gleanings") of Jinadeva to Hemacandra's
[*Abhidhānacintāmaṇi* (ABCH)](abch) — a brief set of further synonyms gathered in the same
kośa form. A small companion to the main kośa.

## When to use it

Use `ACSJ` as a **small companion** to [ABCH](abch) (and its [supplement ACPH](acph)) — gleaned
extra synonyms. For meanings in English use **[Monier-Williams (MW)](mw)**.

## Reading an entry

Brief synonym entries keyed to the kośa's sections. The opening entries
(`csl-orig/v02/acsj/acsj.txt`):

```
<L>1<pc>1
<info kvvv="<s>devADidevakARqaH</s>"/>
<eid>1<syns><s>sarvIya-puM</s>
<eid>4<syns><s>samBava-puM</s>
```

| In the source | Meaning |
|---|---|
| `<info kvvv="…devADidevakARqaH">` | the subject section |
| `<eid>1`, `<eid>4` | entry ids (keyed to the main kośa) |
| `<syns><s>sarvIya-puM</s>` | a gleaned synonym (masculine) |

See [Data Formats](../developers/data-formats) for the markup reference.
