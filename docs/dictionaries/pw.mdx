# BГ¶htlingk, *Sanskrit-WГ¶rterbuch in kГјrzerer Fassung* (`PW`)

After the seven-volume [GroГҹes Petersburger WГ¶rterbuch](pwg), Otto BГ¶htlingk produced a
**shorter recension** on his own вҖ” the *Sanskrit-WГ¶rterbuch in kГјrzerer Fassung* ("in
briefer form"), again in seven volumes, **1879вҖ“1889**. It keeps the headword coverage but
strips much of the citation apparatus and compresses the discussion, making it the more
practical of the two German Petersburg dictionaries. Glosses are in **German**.

:::note Code `PW`, repository `PWK`
On the website and in the source tree this dictionary's **code is `PW`**
(`csl-orig/v02/pw/`), but its GitHub repository is named
[`PWK`](https://github.com/sanskrit-lexicon/PWK). There is no `sanskrit-lexicon/PW` repo.
:::

## When to use it

Use PW when you want **BГ¶htlingk's scholarship in a more compact form** than the
[GroГҹes Petersburger WГ¶rterbuch](pwg): the same authority and German glosses, but condensed.
Go to **[PWG](pwg)** instead when you need the full citation apparatus and the longest sense
discussions, or to **[Monier-Williams (MW)](mw)** for English.

## Reading an entry

PW keeps PWG's markup but trims it: Sanskrit in `{#вҖҰ#}`, German in `{%вҖҰ%}`, abbreviated
sources in `<ls>`, and Greek (used for etymological comparison) in `<gk>`. The privative
prefix *a-/an-* (`csl-orig/v02/pw/pw.txt`):

```
<L>2<pc>1-001-a<k1>a<k2>aЛҡ<h>2
<hom>2.</hom> {#aЛҡ#} vor Consonanten, {#anЛҡ#} vor VocalenВҰ = <gk>бјҖ, бјҖОҪ</gk> <ab>priv.</ab>
{#abrAhmaRa#} {%kein Brahman,%} {#anaDyAya#} {%kein Lesen,%} {#asveda#} {%schweisslos,%} вҖҰ
<LEND>
```

| In the source | Meaning |
|---|---|
| `<pc>1-001-a` | volume-page-column reference (vol. 1, p. 1, col. a) |
| `<hom>2.</hom>` | homonym label |
| `{#aЛҡ#}` | Sanskrit (SLP1); `Лҡ` marks a bound/combining form |
| `<gk>бјҖ, бјҖОҪ</gk>` | Greek, for the etymological comparison |
| `{%kein Brahman,%}` | German gloss |

See [Data Formats](../developers/data-formats) for the full markup reference.

## What makes it distinctive

- **The practical Petersburg dictionary.** Same lemma coverage as PWG, far less apparatus.
- **One author.** BГ¶htlingk alone (Roth was co-author only of the larger work).
- **Etymological notes.** Greek and other comparanda appear inline via `<gk>`/`<lang>`.


## Headword key conventions

`<k1>` is the SLP1 stem (search key); `<k2>` may add `Лҡ` to mark a **combining / bound form** (a prefix or affix never used alone) or carry the udДҒtta `/`. Page references encode **volume-page-column**.

| Convention | Marker / form | Example | Note |
|---|---|---|---|
| Search key | SLP1 stem | `a` | Always search the bare SLP1 form |
| Combining-form mark | `Лҡ` in `<k2>` and `{#вҖҰ#}` | `{#aЛҡ#}`, `{#anЛҡ#}` | Signals the entry is always prefixed, never standalone |
| UdДҒtta (some entries) | `/` on the vowel in `<k2>` | вҖ” | Present in some entries; not typed to search |
| Homonyms | `<hom>N.</hom>` tag | `<hom>2.</hom>` | Labelled as in the original print |
| Page reference | `<pc>vol-page-col` | `<pc>1-001-a` = vol. 1, p. 1, col. a | Three-part locator for the seven-volume set |
