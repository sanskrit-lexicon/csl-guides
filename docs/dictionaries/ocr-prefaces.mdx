# OCR'd Prefaces & Front Matter

Most CDSL dictionaries are scholarly works whose **front matter** — title pages, forewords
(*Vorwort* / *Vorrede* / preface), lists of cited works and abbreviations, and
addenda/corrigenda — contains the editor's own account of sources, method, and conventions.
These pages are **scanned** in [csl-doc](https://github.com/sanskrit-lexicon/csl-doc) but,
until now, not available as searchable text or in translation.

This page indexes a growing effort to **transcribe that front matter faithfully** from the
scans and provide **English and Russian translations** alongside the original. Each
dictionary's output lives in a `prefaces/` directory in its own repository:

- **Per-page files** — `…prefNN.md` (faithful transcription in the source language),
  plus `…prefNN.en.md` and `…prefNN.ru.md` (translations).
- **Consolidated editions** — one file per language with all pages in order and a table of
  contents (`…pref_all.de.md`, `…pref_all.en.md`, `…pref_all.ru.md`).
- **Original scans** under `prefaces/scans/`, and a `README.md` index per dictionary.

Sanskrit (Devanāgarī and transliteration), work titles, bibliographic abbreviation keys,
reference numbers, and Latin quotations are kept **verbatim** across all three languages;
the digitizer's running-header/footer stamps are omitted; printed page numbers are
preserved inline.

**How this is produced:** the `/cologne-preface-ocr` skill (vision band OCR on csldoc
scans). Operator manual, phases (including Phase 3.5 Tesseract audit), and
**Claude Code vs Grok attribution** for every finished dict:
[Preface OCR pipeline](preface-ocr-pipeline).

A separate Tesseract pipeline (`feat/ocr-v2-pipeline` in the [PD](https://github.com/sanskrit-lexicon/PD)
repo) handles Deccan College *Encyclopaedic Dictionary* PDF front matter — not CDSL.

Cologne front-matter index (all codes that have scans):
[csldoc dictionaries index](https://sanskrit-lexicon.uni-koeln.de/scans/csldev/csldoc/build/dictionaries/index.html).

## Queue (not yet on the public / production index)

From the [csldoc front-matter index](https://sanskrit-lexicon.uni-koeln.de/scans/csldev/csldoc/build/dictionaries/index.html), these codes still lack a **production** `prefaces/` deliverable (as of 23-07-2026). The “23 full” list above is only what landed in a real repo; Claude Code left **additional local residue** (see [pipeline manual](preface-ocr-pipeline#claude-code--local-only--incomplete-worktree-residue)).

| Priority | Codes | Notes |
|---|---|---|
| **P0 — promote local-complete (do not re-OCR first)** | GST, PGN, SNP | Full pages + RU + consolidated sit under local `GitHub/prefaces_{gst,pgn,snp}/` only — **not on GitHub remote**; worktree-style staging, easy to lose |
| **P1 — high-use general lexica** | AP90, AP, AE, BEN, BUR | Everyday / historical core; BUR is stub scripts only |
| **P2 — classical / indigenous** | SKD, ABCH | SKD has partial EN excerpts in [skd.mdx](skd) from a docx path, not full scan OCR; SKD `prefaces/scans/` empty |
| **P3 — partial / specialized** | LAN (2/19 pages local), ACC, LRV, PE, PUI | LAN has scans + two pages on disk; PE/PUI stubs |

**Deleted worktrees:** unfinished runs that were never copied into `prefaces_*` or a dict repo leave **no recoverable remote trace**. Operator manual: [Preface OCR pipeline](preface-ocr-pipeline).

Scanned originals for every dictionary remain available through each entry's **csl-doc front-matter
page** (linked from the [catalog](catalog)).

## What's covered (highlights)

- **PWG** — seven-volume St Petersburg dictionary (1855–1875): title pages and forewords per volume, lists of cited works, addenda.
- **PW** — abridged edition, *Erster Theil (Die Vocale)*, 1879: title page, *Vorwort*, abbreviation list.
- **GRA** — Grassmann *Wörterbuch zum Rig-Veda*, 1873: title, foreword, English translation page.
- **MD** — Macdonell, 1893: title, dedication, preface, alphabet, abbreviations.
- **SCH** — Schmidt *Nachträge*, 1928: title + four-page *Vorwort*.
- **MW / MWE / MW72** — Monier-Williams family (1899 SE, English-Sanskrit, 1872).
- **VCP / KRM** — indigenous Sanskrit lexicographic front matter.
- **BOP / STC** — Latin and French sources (Bopp; Stchoupak).

More dictionaries will be added here as their front matter is transcribed via `/cologne-preface-ocr`.
