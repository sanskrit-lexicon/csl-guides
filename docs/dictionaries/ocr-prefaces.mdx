# OCR'd Prefaces & Front Matter

Most CDSL dictionaries are scholarly works whose **front matter** — title pages, forewords
(*Vorwort* / *Vorrede* / preface), lists of cited works and abbreviations, and
addenda/corrigenda — contains the editor's own account of sources, method, and conventions.
These pages are **scanned** in [csl-doc](https://github.com/sanskrit-lexicon/csl-doc) but,
until now, not available as searchable text or in translation.

This page is the **public index of OCR'd front matter**: each dictionary row links the
in-repo `prefaces/` README and the consolidated single-file editions.

- **Per-page files** — `…prefNN.md` (faithful transcription in the source language),
  plus `…prefNN.en.md` / `…prefNN.ru.md` (translations) when needed.
- **Consolidated editions** — one file per language with all pages in order and a table of
  contents (`…pref_all.de.md`, `…pref_all.en.md`, `…pref_all.ru.md`, …).
- **Original scans** under `prefaces/scans/` where the source is multi-page PNG/JPG scans.
- **Operator manual** (phases, engines A/B, attribution): [Preface OCR pipeline](preface-ocr-pipeline).
- **Sample quality probe (PWG/PW, H1559):** [OCR + EN error-rate report](https://github.com/sanskrit-lexicon/csl-guides/blob/main/scripts/out/H1559_pref_ocr_en_sample_error_rates_24.07.2026.md) — stratified 15 pages, scan-locus + dual-pass + EN sense tables.

Sanskrit (Devanāgarī and transliteration), work titles, bibliographic abbreviation keys,
reference numbers, and Latin quotations are kept **verbatim** across languages; digitizer
running-header/footer stamps are omitted.

**How this is produced:** `/cologne-preface-ocr` — vision band OCR on csldoc scan pages, or
HTML text-layer extract when csldoc already has a transcribed single page (AP90, AE, BEN, ACC).
A separate Tesseract pipeline in [PD](https://github.com/sanskrit-lexicon/PD) (`feat/ocr-v2-pipeline`)
covers Deccan College *Encyclopaedic Dictionary* PDF front matter — not CDSL.

Cologne scan index (all codes that have front-matter pages):
[csldoc dictionaries](https://sanskrit-lexicon.uni-koeln.de/scans/csldev/csldoc/build/dictionaries/index.html).
