# Meulenbeld, *Sanskrit Names of Plants* (`SNP`)

`SNP` is G. J. Meulenbeld's *Sanskrit Names of Plants* — a specialist resource that
**identifies Sanskrit plant names**, mapping each to its botanical equivalent(s) and the
lexical and scientific **sources** that record it. Indispensable for reading āyurvedic and
botanical texts.

## When to use it

Reach for `SNP` to **identify a Sanskrit plant name** — its synonyms/botanical identity and the
sources that attest it — when reading āyurveda, dharmaśāstra, or kāvya. For general meanings use
**[Monier-Williams (MW)](mw)**.

## Reading an entry

Each entry is a **plant name** (`<k1>`) with equivalences and a dense list of source
abbreviations. The first entry (`csl-orig/v02/snp/snp.txt`):

```
<L>1<pc>521<k1>akza<k2>akza
{%akṣa%}¦
<div n="P"/>(1) = {%vibhītaka%} (Avk; Bpn, p. 9; Dgv, nr. 149; Dn 1, 212; Gul; HB;
HK; KB 2, p. 1017-1020; MW; Pr 220; PW; Rn 11, 322-323; V 6, p. …
```

| In the source | Meaning |
|---|---|
| `{%akṣa%}` | the **plant name** (search key `<k1>`) |
| `(1) = {%vibhītaka%}` | the identification (here = *vibhītaka*, the beleric myrobalan) |
| `(Avk; … MW; … PW; …)` | the **sources** attesting it (incl. MW and PW) |

See [Data Formats](../developers/data-formats) for the markup reference.

## What makes it distinctive

- **Botanical identification** of Sanskrit plant names, with a thorough source apparatus.


## Headword key conventions

`<k1>` = `<k2>` = SLP1 (Sanskrit plant name). Each entry maps the name to its botanical identification(s) and the lexical or scientific sources that record it.

| Convention | Marker / form | Example | Note |
|---|---|---|---|
| Search key | SLP1 (plant name) | `akza` | k1 = k2 |
| Body form | IAST italic `{%…%}` | `{%akṣa%}` | The printed plant name |
| Identification | `(N) = {%botanical name%}` | `(1) = {%vibhītaka%}` | A numbered botanical equivalent |
| Sources | Abbreviated list in parentheses | `(MW; PW; Avk; …)` | Lexical + scientific sources attesting the identification |
