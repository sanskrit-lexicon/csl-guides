# Events & Activity

The Cologne Digital Sanskrit Dictionaries is a **data and lexicon project** — not a lab or
event series. It does not host its own talks, workshops, or conferences, so there is no
standing events calendar.

What the project keeps instead is a **running activity record**:

- **[csl-newsletter](https://github.com/sanskrit-lexicon/csl-newsletter)** — a *daywise log*
  of changes and additions (September 2021 onward), kept "so that an academic paper may be
  published later, if desired or required."
- **[News](/news)** — release notes and milestones for these guides and for the
  dictionaries.
- **[GitHub activity](https://github.com/sanskrit-lexicon)** — issues, pull requests, and
  releases across the organization.

If CDSL work is presented at a venue, or a related workshop is organized, it would be
recorded in the newsletter and (for anything user-facing) in [News](/news).
